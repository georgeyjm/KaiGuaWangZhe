### 安装

注：本程序开发在 macOS 10.13.4 系统，Python 版本 3.5

1. 安装网络抓包程序（推荐使用 Charles）

2. *(可选)* 安装 MongoDB

3. 安装 Python 模组：

```
pip install requests jieba bs4 watchdog selenium pymongo
```

4. 下载本代码仓库：

```
git clone https://github.com/yu-george/KaiGuaWangZhe.git
```
